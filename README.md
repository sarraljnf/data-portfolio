# Data Science & Artificial Intelligence Portfolio
**By Sarra Lajnef**

This repository is a comprehensive collection of my work in Data Science, Machine Learning, and Operational Research. It transitions from fundamental theory (found in the Metrics Lab and SQL sections) to complex, real-world applications in Healthcare and Semantic Segmentation.

---

### Featured Project: AI in Healthcare (Intelligence Artificielle et Santé)
**Goal:** Optimizing hospital resource management by predicting and planning patient appointments using a combination of Machine Learning and Operational Research.
* **Key Components:**
    * **Preprocessing:** Cleaning patient datasets.
    * **Predictive Modeling:** Generating patient flow predictions.
    * **Operational Research:** Appointment attribution and scheduling optimization (using Genetic Algorithms and SMA).
* **Technologies:** Python, Machine Learning, Optimization Algorithms.

---

### Deep Learning & Advanced Research
* **Semantic Segmentation:** In-depth study on Fourier Domain Adaptation (FDA) for semantic segmentation, including detailed notes and research papers.
* **Time-Series Specialty:** Advanced time-series analysis covering Linear Processes (AR/MA), Deep Learning for forecasting, Anomaly Detection, and Hypothesis Testing.
* **Machine Learning Fundamentals:** Practical implementations of Supervised Learning (Classification/Regression) and Unsupervised Learning (Clustering).

---

### Data Engineering & Analytics
* **The Metrics Lab:** A specialized environment for evaluating model performance through detailed Classification, Regression, and Clustering metrics.
* **SQL & Data Analytics:** Collection of complex SQL queries and data analysis workflows for structured database management.

---

### Repository Structure
* **Application/**: Healthcare project (Intelligence Artificielle et Santé) including code, data, and reports.
